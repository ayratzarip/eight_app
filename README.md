# Logbook

**Logbook** — приложение для ведения журнала самонаблюдения и работы с шагами к цели. Два основных раздела: **Журнал** (структурированные записи ситуаций) и **Шаги к цели** (список шагов с отметкой выполнения).

## Описание

Logbook помогает структурированно анализировать социальные ситуации, отслеживать фокус внимания, мысли, телесные ощущения и действия, планировать следующие шаги. Модуль «Шаги к цели» позволяет формулировать и отмечать выполнение шагов к личным целям.

## Основные возможности
- Ведение журнала самонаблюдения с пошаговым анализом ситуаций (ситуация, фокус внимания, мысли, телесные ощущения, действия, планы)
- Шаги к цели: добавление, редактирование, удаление, изменение порядка (drag&drop), отметка выполнения
- Подсказки по заполнению каждого шага формы записи (кнопка «Подсказка»)
- Экспорт записей журнала в CSV и копирование в буфер для AI
- Экран инструкций: как заполнять журнал, советы, формирование шагов к цели
- Современный дизайн с поддержкой светлой и тёмной темы
- Локальное хранение данных (без передачи в интернет)

## Модули

### 1. Журнал самонаблюдения
- Пошаговая форма записи: дата и время, описание ситуации, фокус внимания, мысли, телесные ощущения (с шкалой интенсивности 0–10), действия и результат, планы на будущее
- Подсказки по заполнению каждого шага (всплывающее окно по кнопке «Подсказка»)
- Список записей с поиском
- Просмотр, редактирование и удаление записей
- Экспорт в CSV и копирование данных для AI

### 2. Шаги к цели
- Список шагов к цели с отметкой выполнения (чекбокс)
- Добавление, редактирование, удаление шагов; изменение порядка перетаскиванием
- Разделение на «Следующий шаг» и «Завершённые шаги»
- Поиск по шагам
- Инструкции по формированию шагов и советы (на экране «Инструкции»)

## Техническая реализация
- Flutter, Dart
- Provider для управления состоянием
- Хранение: SQLite для записей журнала и шагов к цели на мобильных платформах; на веб-платформе — SharedPreferences (данные в JSON)
- Модель целей с json_serializable (Goal, goal.g.dart)
- Архитектура: модели, сервисы, провайдеры, экраны, виджеты, общие стили (app_styles)

## Дизайн
- Material Design 3, шрифт Inter (Google Fonts)
- Скруглённые карточки, тени, единая палитра (AppColors, AppRadius, AppButtonStyles и др.)
- Поддержка светлой и тёмной темы
- Раздельная цветовая схема для журнала (logoGreen) и шагов к цели (goalsScreen)

## Приватность и безопасность
- Все данные хранятся только локально на устройстве
- Нет передачи данных в интернет
- Пользователь полностью контролирует свои записи и цели

## Планы развития
- Связь шагов к цели с записями журнала
- Напоминания о записях и шагах
- Категории и теги
- Экспорт шагов к цели в CSV
- Расширенная аналитика
